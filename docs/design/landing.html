Как это работает

Три шага от регистрации до работающей поддержки — без внедренцев и месяцев настройки.

1 Подключите каналы

Web SDK вставляется одной строкой, Telegram-бот подключается по токену. Первое сообщение — через минуты.

2 Настройте смену

Роли, лимиты операторов, тематики и SLA — очередь сама распределяет и спасает просроченные диалоги.

3 Контролируйте качество

Оценки клиентов, проверки текста, отчёты с XLSX-выгрузкой и журнал аудита каждого действия.

Частые вопросы

Не нашли ответа — напишите нам, отвечаем в рабочие часы в тот же день.

Сколько занимает запуск?

Обычно один день: регистрация создаёт рабочую организацию, SDK ставится одной строкой, Telegram подключается по токену бота.

Какие каналы поддерживаются?

Web SDK, Telegram, ВКонтакте, MAX и REST API уже работают в продакшене. WhatsApp, Email и Viber — на подключении.

Как устроен trial?

Trial стартует с полноценной организацией: первый администратор, лимиты операторов и тестовое сообщение. Карта не нужна.

Как защищены данные?

Email OTP, роли и блокировки, изоляция организаций, маскирование PII и полный журнал аудита с экспортом.

Можно ли выгружать отчёты?

Да — отчёты по диалогам, первому ответу, SLA и назначениям выгружаются в XLSX; аудит доступен через API.
